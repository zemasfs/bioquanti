23.1 Variação de parâmetro em função plotável e output de resultado

      Uma animação com uso de laço iterativo (loop) sem qualquer pacote adicional pode agregar valor à compreensão das relações entre variáveis, tal como segue.

## Relação de pH com potencial de equilíbrio (Nernst)

## Parâmetros da equação
Eo = 0.7 # potencial padrão, V
R = 8.314 # const. geral dos gases, J/mol/K
T = 298 # temperatura absoluta
F = 96485 # const. de Faraday, C/mol
## Vetor de no. de elétrons
n = seq(from=1,to=5,by=1)

## Iteração para a equação de Nernst com variação em "n"
for (i in 1:length(n)) {
  add <- if(i==1) FALSE else TRUE
  curve(Eo-2.303*(R*T/(n[i]*F))*x+runif(length(n),0,0.02),
        type = "b",col= i, lwd=0.8, 
        from =1 , to=14, n =10, cex =0.5,
        xlab="pH", ylab="E, V", add=add)
  nernst.slope <- 2.303*(R*T/(n[i]*F))
  cat("inclinação de Nernst (mV/pH): ", round(nernst.slope*1e3,digits = 1),"\n")
  add<-if(i == 1) FALSE else TRUE # controle de fluxo que permite adição de curva a partir da segunda iteração (ou seja, quando i > 1)
  Sys.sleep(0.5) # pausa entre as interações
}

23.2 O pacote plotly

      O exemplo abaixo apresenta um gráfico no qual características dos dados podem ser visualizadas por arraste de mouse sobre os mesmos (mouse over).

tempo <- c(rep(rep(1:4, 4),4))
resposta <- c(1,2,3,4,2,4,6,8,10,14,18,22,24,30,36,42)
niveis <- c(rep("A",4),rep("B", 4), rep("C", 4), rep("D", 4)) # cria vetores com níveis

df <-data.frame(tempo, resposta, niveis) # cria um *dataset* multivariado

library(ggplot2)
library(plotly) # carrega as bibliotecas necessárias

ggplotly(p) # executa uma animação, no caso, um *mouse over* que fornece os dados de cada ponto do gráfico

23.3 O pacote manipulate

      Entre outras características, é possível a criação de um controle deslizante (slider) para que se observe o resultado gráfico sobre a variação de um ou mais parâmetros de uma equação.

      Exemplificando, uma curva hiperbólica com variação no parâmetro de meia resposta (k), como segue. Para obter o resultado visual, é necessário rodar o trecho de código no RStudio.

library (manipulate)
k=1 # valor inicial do parâmetro a variar
x <- 1:100 # criação de vetor
manipulate(plot(x,2*x/(k+x),
                ylab="f(x)", ylim=c(0,2)),
                k = slider(0.1, 20)) # aplicação da função

# Obs: o resultado só pode ser visualizado dentro do *RStudio*

      O mesmo exemplo acima pode ser também obtido por uma função matemática, e não apenas por plotagem de pontos, como segue.

library(manipulate)
manipulate(curve(2*x/(k+x),
                 ylab="f(x)", from=0, to=50,
                 ylim=c(0,2)), k = slider(0.1, 50))

# Obs: o resultado só pode ser visualizado dentro do *RStudio*

23.5 O pacote iplots

      Apesar de antigo, o pacote iplots permite a criação de gráficos interativos (pontos, barras, histogramas e mosaico) que destacam os valores de um dado em mais de um gráfico. Isso é particularmente útil quando se deseja identificar um determinado ponto de um gráfico cujas variáveis são transformadas para a produção de outro. Exemplificando do pacote:

data(iris)
attach(iris)
iplot(Sepal.Width,Petal.Width)
iplot(Sepal.Width/Sepal.Length, Species)

      O pacote também permite ampliação (zoom) e alguns efeitos adicionais (brushing).
