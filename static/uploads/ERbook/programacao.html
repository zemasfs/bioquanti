24.2 Switch

        Trata-se de uma função alternativa para sequência de IF concatenados, sendo mais rápido e legível.

switch ( condição , lista ) # onde "lista" é caso 1, caso 2,...

      A função é flexível, podendo apresentar como saída um valor/caractere indexado numa lista:

switch( 3, "arroz", "feijão", "agrião", "couve", "macarrão" )
[1] "agrião"
numer <- round(sqrt(rnorm(1, mean = 2, sd = 1)^2)) # cria número aleatório próximo de 1, 2, e em módulo
switch( numer, { print( "Acima da média." ) }, {
print( "Abaixo da média." ) } ) 
[1] "Abaixo da média."

      Alternativamente, pode-se aplicar um trecho de código selecionado a partir da indexação na lista:

qualidade <- "regular"
switch(qualidade,
       "péssimo" = {qualidade <- 4},
       "ruim" = {qualidade <- 6},
       "mediano" = {qualidade <- 7},
       "regular" = {qualidade <- 8},
       "bom" = {qualidade <- 9},
       "ótimo" = {qualidade <- 9.5},
       "excelente" = {qualidade <- "um semideus!"})
print(paste("O produto apresentou qualidade abaixo de", qualidade ))
[1] "O produto apresentou qualidade abaixo de 8"

      Embora o comando switch opere com caracteres (strings), pode-se contornar uma solução de ranqueamento numérico acima, como segue:

qualidade <- "7"
switch(qualidade,
       "2" = {qualidade <- "péssima"},
       "4" =  {qualidade <- "ruim"},
       "6" =  {qualidade <- "mediana"},
       "7" = {qualidade <- "regular"},
       "8" = {qualidade <- "boa"},
       "9" = {qualidade <- "ótima"},
       "10" = {qualidade <- "excelente"})
print(paste("O produto apresentou ", qualidade, "qualidade."))
[1] "O produto apresentou  regular qualidade."

      O comando também permite a seleção de uma função, trecho de código, ou cálculo matemático, esse abaixo exemplificado:

param1 = 5
param2 = 12
escolha = "b"
saida = switch(escolha,
  "a"= cat("(p1+p2)^2 =", (param1 + param2)^2),
  "b"= cat("(p1+p2)^3 =", (param1 + param2)^3),
  "c"= cat("(p1+p2)^4 =", (param1 + param2)^4))
(p1+p2)^3 = 4913
cat(" . Essa foi a equação encontrada e seu valor.", saida)
 . Essa foi a equação encontrada e seu valor.

24.3.2 FOR

      O laço for talvez seja o mais empregado em iterações para cálculos e gráficos, posto que é aplicado nos principais objetos a esses correlatos, como vetores, matrizes de dataframes.

for ( variável in sequência ) 
{
execute
}

      A sequência pode ser prefixada, como “1:4”, ou definida em função do comprimento do objeto, como num vetor, “1:length(vetor). Ilustrando para uma operação no próprio índice:

for( j in 1:4 )
{
print( paste( "Quadrado do índice ", j, "é de", j^2 ) )
}
[1] "Quadrado do índice  1 é de 1"
[1] "Quadrado do índice  2 é de 4"
[1] "Quadrado do índice  3 é de 9"
[1] "Quadrado do índice  4 é de 16"

      E para uma operação em vetor:

vetor <- c( 2, 4, 8, 12 )
for( j in 1:length(vetor) )
{
print( paste( "Quadrado do índice ",vetor[j], "é de", vetor[j]^2 ) )
}
[1] "Quadrado do índice  2 é de 4"
[1] "Quadrado do índice  4 é de 16"
[1] "Quadrado do índice  8 é de 64"
[1] "Quadrado do índice  12 é de 144"
x <- 1:5
y <- x^2 + runif(5) # simula uma correlação 
lmxy <- lm(y~x) # perfaz a regressão linear
saida <- list( A=coef(lmxy)[1], B=coef(lmxy)[2], C=cor(x,y)) # saída como valores de intercepto, inclinação, e coeficiente de correlação linear

for( i in saida ) # loop
{
  print(paste(i))
}
[1] "-6.48627170193939"
[1] "5.96198708796874"
[1] "0.982105748419805"
