5  Cabeçalho

      O cabeçalho de um documento em Rmarkdown ou Quarto, ou de um projeto (nesse caso, pelo arquivo de configuração *_quarto.yml*) direciona como o documento será formatado. Um cabeçalho mínimo pode ilustrar-se por:

---
title: "Documento"
---

      O que basta para começar digitar o documento. De fato, nem mesmo esse dado é necessário, embora seja informativo à identificação do material em compilação. Mas existem diversar outras formatações que se extendem ao documento como um todo, e mesmo para um projeto (livro, website), e das quais mencionam-se algumas abaixo.
      É importante se observar a identação correta do cabeçalho (ou do arquivo de metadados *_quarto_yml*), pois uma identação errada não permitirá compilar o documento.

date: "`r Sys.Date()`" # data automática

      Para a criação de um documento com numeração de seções com três subníveis, por exemplo, basta:

---
title: "Documento"
number-sections: true
number-depth: 3
---

      A saída do documento (output) pode ocorrer como HTML (Hypertext Markup Language), PDF (Portable Document Format) ou DOCX (Microsoft-Word). Para isso, basta escolher:

---
title: "Documento"
format: html # ou pdf ou docx

      Um exemplo mais completo (embora longe de abranger as todas as possibilidades) é dado abaixo para a elaboração de um livro.

# Principal
project:
  type: book
  output-dir: _book
editor: visual
title: "Meu livro"
reader-mode: true
author: "euzinho"
date: today
date-format: "D, MMM, YYYY"
lang:pt

# Bibliografia e formatação
bibliography: references.bib
csl: american-medical-association.csl
classoption:
  - twocolumn
  - landscape
papersize: a5
linestretch: 1.5   ; espacejamento
fontsize: 12pt
links-as-notes: true

# Output
format:
  html:
    theme: cosmo
  pdf:
    documentclass: scrreprt
    includes:
      in_header: "preamble.tex"
    fontfamily: accanthis
    latex_engine: pdflatex

  docx: default
  html:
    css: styles.css
  pdf:
    documentclass: report
    margin-left: 30mm
    margin-right: 30mm

# Rodapé, repositório Git, e direitos autorais
page-footer:
    left: |
      Meu Livro foi escrito por Mim
    right: |
      Este livro foi construído com auxílio de <a href="https://quarto.org/">Quarto</a>.
  cover-image: cover.jpg
  favicon: cover.jpg
  site-url: https://meusite.br
  repo-url: https://github.com/euzinho/meuLivro/
  repo-branch: main
  repo-actions: [edit, issue]

  # Conteúdo (partes e capítulos)
  chapters:
    - index.qmd
    - a.qmd
    - part: b.qmd
    chapters:
        - c.qmd
        - d.qmd
        - e.qmd
    - part: f.qmd
    chapters:
        - g.qmd
        - h.qmd
        - i.qmd
