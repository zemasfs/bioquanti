  2. Figura
  3. Gráfico
  4. Tabela
  5. Cronograma
  6. Fluxograma / Diagrama

  9. Análise de dados
  10. Cálculos
  11. Pacotes de LaTex
 tamanhos de etiquetas, espessura, e valores dos eixos

- criar paineis

o que uma pessoa precisa após obter o dado
carregar os dados: compilar o dado numa planilha…ou analisá-lo diretamente no R
plotar pra ver a relação entre variáveis (plot, pairs)
acrescentar linha pontilhada em determinado marco
analisar os dados: sumário estatístico, diferenças de médias/variâncias, testes pos-hoc
plot de subconjuntos mais desejados
simular um cenário e observar parâmetros de uma equação
produzir gráficos de qualidade pra publicação

Pra carregar os dados:
copiando e colando 
criando
na unha
colando de planilha
juntando vetores
criando dataframe ou matriz
importando (Excel, csv)

Pra fazer um plot:
Seleciona as variáveis de um dataset (ou faz o plot de um contendo apenas x e y), ou seleciona um subconjunto desejado, ou seleciona alguma estatística dos dados (mean e sd de triplicatas - 3 vetores y, por ex)
Processa dos dados
pra evitar missing data (na.rm=TRUE)
pra transformar variáveis
Escolhe um tipo de plot (pontos, linhas, barras)
Faz o plot
Se tiver barras de erros, já coloca antes (ou depois)
Coloca (ou não) legenda
Melhora a visualização (símbolo, tamanho, cor, fonte, eixos)
Altera as etiquetas de eixos
Adiciona outro eixo, se necessário
Escreve algo no plot
Adiciona uma linha de marcação no plot -  (abline(h=0))
Altera plano de fundo
Verifica relação entre variáveis (modelagem)- linear, polinomial, curvilinear (não linear)
Obtém os parâmetros do modelo
Informa no plot o modelo/parâmetros
Refina o plot pra publicação
